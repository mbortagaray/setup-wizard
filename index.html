👋 Bem-vindo!
Vamos configurar suas credenciais Google OAuth. Este processo é necessário para que sua automação possa acessar os serviços do Google.

O que você vai precisar:

  1. Uma conta Google (Gmail)
  2. Acesso ao Google Cloud Console
  3. 10-15 minutos do seu tempo

O que vamos fazer:

  1. Criar um projeto no Google Cloud
  2. Ativar as APIs necessárias
  3. Configurar a tela de consentimento
  4. Gerar as credenciais OAuth
📁 Passo 1: Criar Projeto no Google Cloud
Primeiro, você precisa criar um projeto no Google Cloud Console. Um projeto organiza todos os recursos do Google Cloud.
🔗 Abrir Google Cloud Console - Criar Projeto

Instruções:

  1. Clique no link acima
  2. Clique em "CREATE PROJECT" (Criar Projeto)
  3. Digite um nome para o projeto (ex: minha-automacao)
  4. Em "Organization", selecione "No organization"
  5. Clique em "CREATE"
  6. Aguarde a confirmação verde no topo

O ID aparece na tela do projeto ou na barra de endereço

🔗 Passo 2: Ativar APIs do Google
Agora você precisa ativar as APIs que sua automação vai usar. Cada API dá acesso a um serviço diferente.
🔗 Abrir Biblioteca de APIs

Selecione seu projeto:

  1. No menu superior esquerdo, clique no seletor de projeto
  2. Selecione o projeto que você acabou de criar
  3. Você verá o nome do projeto no topo da página

Ativar APIs (escolha as que sua automação precisa):

  1. Pesquise e ative Gmail API (para automações de email)
  2. Pesquise e ative Google Sheets API (para planilhas)
  3. Pesquise e ative Google Drive API (para arquivos)
  4. Clique na API → "ENABLE"
🖥️ Passo 3: Configurar Tela de Consentimento
A tela de consentimento aparece quando alguém tenta fazer login com sua conta. Vamos configurar isso.
🔗 Abrir Tela de Consentimento OAuth

Instruções:

  1. Selecione seu projeto no topo
  2. Selecione "External"
  3. Clique "CREATE"
  4. Preencha:
    • App name: Automacao n8n
    • User support email: seu email
  5. Clique "SAVE AND CONTINUE"
  6. Na página de Scopes, clique "ADD OR REMOVE SCOPES"
  7. Selecione as scopes (permissões) que você precisa
  8. Clique "UPDATE" → "SAVE AND CONTINUE"
  9. Adicione seu email como Test User
  10. Clique "SAVE AND CONTINUE"
🔑 Passo 4: Criar Credenciais OAuth
Agora vamos criar o arquivo de credenciais que você vai usar no n8n.
🔗 Abrir Credenciais

Instruções:

  1. Selecione seu projeto no topo
  2. Clique "+ CREATE CREDENTIALS"
  3. Selecione "OAuth client ID"
  4. Em "Application type", selecione "Web application"
  5. No campo "Name", digite: Automacao-n8n
  6. Clique "CREATE"

⚠️ Importante:

Após clicar CREATE, vai aparecer um popup com seu Client ID e Client Secret. Você também pode clicar em "Download JSON".

Copie ambos ou baixe o arquivo!

📤 Passo 5: Enviar Credenciais
Almost lá! Cole o conteúdo do arquivo JSON de credenciais abaixo para finalizar.

Como obter o JSON:

  1. Se você já baixou o JSON: abra no Bloco de Notas e copie tudo
  2. Se não baixou: copie o Client ID e Client Secret do popup

Se preferir, pode colar apenas o Client ID e Client Secret nos campos acima
